## How To Run

Each `.java` file is independent and can be compiled or run on its own.

```bash
javac Basics/HelloWorld.java
java Basics.HelloWorld
```

If you are running files without packages, you can also compile and run from inside the `Basics` folder:

```bash
cd Basics
javac HelloWorld.java
java HelloWorld
```
